
## 프로그램 리스트
- 자동 시작 설치 프로그램 (autorun_install.exe, autorun_install_xp.exe)
- 자동 분석 설치 프로그램 (medical_analyzer_install.exe)


## A. 자동 시작 설치 프로그램

### A.1. 설치 방법
#### 아래에서 운영체제에 맞는 파일을 클릭해서 다운로드 하세요
 * Windows XP 용 : [medical_autorun_install_xp.exe](https://github.com/winnyhs/release/releases/download/autorun_latest/medical_autorun_install_xp_v1.0.0.exe)
 * Win7, Win10 용: [medical_autorun_installer.exe](https://github.com/winnyhs/release/releases/download/autorun_latest/medical_autorun_install_v1.0.0.exe)

#### 다운받은 파일을 더블클릭하세요.
   * 2초후 설치가 완료 됩니다.
   * 컴퓨터를 재시작하면 프로그램 읽기 창까지 자동으로 진행됩니다. 
   * 컴퓨터를 재시작하지 않고, 수동으로 실행하고 싶은 경우를 위에 바탕화면에 자동 읽기, 자동 실행 2개의 아이콘이 있습니다. 

### A.2. 사용 방법
* 사용자 매뉴얼 참고



## B. 자동 분석 설치 프로그램
* Win7 이상에서 동작

### B.1. 설치 방법

1. 아래 2 개의 파일을 모두 다운로드 하세요
  * [tesseract-ocr-setup-3.05.01-20170602.exe](https://github.com/winnyhs/release/releases/download/analyzer_latest/tesseract-ocr-setup-3.05.01-20170602.exe)
  * [medical_analyzer_install.exe](https://github.com/winnyhs/release/releases/download/analyzer_latest/medical_analyzer_install_v1.0.0.exe)

2. 다운받은 tesseract-ocr-setup-3.05.01-20170602.exe 를 더블클릭해서 설치하세요 
   유의사항) 설치 디렉토리 변경하지 말라

3. 다운받은 medical_analyzer_install.exe 를 더블클릭해서 설치하세요
   * 안내에 따라 Install 버튼을 클릭하기만 하면, 설치가 진행됩니다.
   * Finish 버튼을 클릭하세요. 설치가 완료되었고, 설치 창이 종료됩니다. 
   * 바탕화면에 Medical Analyzer Server 아이콘이 생성되어 있습니다.  

### B.2. 설치 후 사용법

#### 시작하기

1. 바탕화면에 생성된 Medical Analyzer Server를 더블클릭 하세요.  
2. 검은 색 창들이 2개가 뜹니다. 
   검은 창들을 최소화해도 됩니다. 다만, 절대로 종료하면 안 됩니다.
   (종료하면 기능이 중단됩니다.)  
3. 크롬, 엣지, 혹은 익스플로러가 뜨고 고객 정보 페이지가 뜹니다.
   * 고객 정보 페이지에서 +고객을 클릭해서 신규 고객을 등록하세요.
   * 분석 실행 페이지에서 등록된 고객을 선택하고 분석 시작을 클릭해서 분석을 시작하세요.
     * 분석이 완료되면 뜨는 창에서 처방 내용을 검토하세요.
     * 고객에게 기본 프로그램이 다 수행될 때까지 처방 입력을 하지 마세요.
       * 처방 입력을 하면 프로그램이 모두 종료됩니다. 
   * 처방 입력 페이지에서 분석일자를 선택하고 처방저장을 클릭해서 처방 프로그램을 입력하세요. 


#### 종료하기
```bash
1. 검은 색 창들을 모두 닫는다

   기능이 더 이상 동작하지 않는다.  
    (크롬, 엣지, 익스플로러 에서 기능을 클릭하면 동작하지 않는다)

2. 자세한 사항은 사용자 매뉴얼 참고
```
