

# 자동 시작 프로그램 설치

## XP 설치 절차
1. 아래 파일을 클릭해서 다운로드 하세요

   [medical_autorun_install_xp.exe](https://github.com/winnyhs/release/releases/download/autorun_latest/medical_autorun_install_xp_v1.0.0.exe)
   
3. 다운로드 받은 파일을 더블 클릭하세요.  
   * 2초후 설치가 완료 됩니다.
   * 컴퓨터를 재시작하면 프로그램 읽기 창까지 자동으로 진행됩니다. 
   * 컴퓨터를 재시작하지 않고, 수동으로 실행하고 싶은 경우를 위에 바탕화면에 자동 읽기, 자동 실행 2개의 아이콘이 있습니다.


## Windows 7, Windows 10 설치 절차
1. 아래 파일을 클릭해서 다운로드 하세요

   [medical_autorun_installer.exe](https://github.com/winnyhs/release/releases/download/autorun_latest/medical_autorun_install_v1.0.0.exe)

2. 다운받은 파일위에 마우스 우클릭하고 "관리자 권한으로 실행"을 선택하세요.
   * 권한 관련 창이 뜰 수 있습니다. 확인을 선택하세요. 
   * 2초후 설치가 완료 됩니다.
   * 컴퓨터를 재시작하면 프로그램 읽기 창까지 자동으로 진행됩니다. 
   * 컴퓨터를 재시작하지 않고, 수동으로 실행하고 싶은 경우를 위에 바탕화면에 자동 읽기, 자동 실행 2개의 아이콘이 있습니다.
